css-find
===========

Usage:

  Usage: css-find [options] [file path | url] [css prop]

  Example:
    css-find  ./index.css z-index
    css-find  http://www.douban.com z-index

  Options:

    -l, --values-with-matches <path>   Only print values that don't contain selectors
